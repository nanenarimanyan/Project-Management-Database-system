# Project Management Database System

A PostgreSQL-based database system designed for managing projects, employees, tasks, milestones, expenses, and project risks.

This project was developed as a final database systems project and focuses on relational database design, normalization, query optimization, and project analytics.

## Features

- Project and client management
- Employee and department tracking
- Task and milestone organization
- Expense and risk monitoring
- Time log management
- Analytical SQL queries and reports

## Technologies

- PostgreSQL
- SQL
- pgAdmin
- Python

## Database Overview

The system contains more than 20,000 generated records across multiple related tables, including:

- Clients
- Projects
- Employees
- Departments
- Tasks
- Milestones
- Time Logs
- Project Expenses
- Project Risks

## Concepts Used

- Primary and foreign keys
- Normalization
- One-to-many and many-to-many relationships
- Views
- Stored procedures
- Query optimization
- Analytical reporting

## Files

-  DDL.sql → database structure
-  PYTHON_FAKER1.py  → python code to generate data and DML.sql
-  DML.sql → All table content


## Sample Use Cases

- Tracking project progress
- Monitoring employee workload
- Calculating project expenses
- Generating performance reports
- Risk analysis and management
